## Search Console Submission Instructions

### Google Search Console
1. Go to https://search.google.com/search-console
2. Add property: `https://aivedha.ai`
3. Navigate to Sitemaps
4. Submit: `https://aivedha.ai/sitemap-index.xml`

### Bing Webmaster Tools
1. Go to https://www.bing.com/webmasters
2. Add site: `https://aivedha.ai`
3. Navigate to Sitemaps
4. Submit: `https://aivedha.ai/sitemap-index.xml`

### Yandex Webmaster
1. Go to https://webmaster.yandex.com
2. Add site: `https://aivedha.ai`
3. Submit sitemap: `https://aivedha.ai/sitemap-index.xml`

### Baidu Webmaster (China)
1. Go to https://ziyuan.baidu.com
2. Add site: `https://aivedha.ai`
3. Submit sitemap: `https://aivedha.ai/sitemap-zh.xml`
